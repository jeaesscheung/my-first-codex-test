# my-first-codex-test

这个仓库现在包含两个独立的 GitHub Pages 前端项目：

## 项目入口

- **英语对答 APP**：`english-app/`
  - GitHub Pages 路径：`/my-first-codex-test/english-app/`
  - 本地入口：打开 `english-app/index.html`

- **AI 影视中台**：`platform/`
  - GitHub Pages 路径：`/my-first-codex-test/platform/`
  - 本地入口：打开 `platform/index.html`

仓库根目录的 `index.html` 是双项目入口页，提供两个大按钮分别跳转到上述项目。

## 目录说明

```text
/
├── index.html          # 双项目入口页
├── english-app/        # 当前英语对答 APP
├── platform/           # 从 Git history 恢复的原 AI 影视中台
└── .github/workflows/  # GitHub Pages 部署配置
```

## GitHub Pages

部署完成后，可以通过以下路径访问：

- `https://<username>.github.io/my-first-codex-test/english-app/`
- `https://<username>.github.io/my-first-codex-test/platform/`
